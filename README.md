Она принимает:
*arr - указатель на массив.
size - количество элементов в массиве.

Minval - минимальное значение, в начале присвоим первый элеиент массива.

Далее идёт поиск минимума: перебираем элементы массива от i, пока i меньше размера массива, если текущий элемент от i меньше максимального, то присваем минимальному значению значение текущего элемента.

В конце функция возвращает найденный максимум.
